Trace a printf() style message to channel 32.

Formats a printf() style trace message and outputs it to trace channel 32. This function does not support floating point numbers. On this channel, no headers (except for the uint8_t channel number) exist. Thus the function effectively prepends a space (' ') and appends a CR ('') to the trace message.
